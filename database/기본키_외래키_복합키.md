#  기본키, 외래키, 복합키

## 기본키 (Primary Key)
- 후보키들 중에서 하나를 선택한 키로 최소성과 유일성을 만족하는 속성
- 테이블에서 기본키느 오직 하나만 지정 가능
- 절대로 NULL 값을 가질 수 없고 중복된 값을 가질 수 없다.
